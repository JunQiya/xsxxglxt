### 启动项目

```bash
# 切换镜像
npm config set registry https://registry.npm.taobao.org

# 安装依赖
npm install

# 更新依赖
npm update

# 启动项目
npm run dev
#
localhost 改为服务器后端IP地址
比赛端口IP地址：124.221.100.59

# 运行前记得打开redis缓存
# 综合服务平台里的跳转为124.221.100.59及其他服务器地址，暂未做修改
```
